# Fallout-4-Next-Gen-V1
Un nuevo proyecto para Fallout 4 que busca combinar una jugabilidad y experiencia visual de esta década, siendo amigble mayormente para jugadores casuales de habla hispana

Requisitos de instalación:
- Instalar maanulmente F4SE https://www.nexusmods.com/fallout4/mods/42147
- Instalar manualmente ENB Series v501 http://www.enbdev.com/mod_fallout4_v0501.htm
- Instalar manualmente Reactor ENB y todos sus requisitos (unicamente los que aparecen en la sección "Requisitos" de Nexus) y dentro del juego configurar el menú de NACX como lo indica la descripción de Reactor
